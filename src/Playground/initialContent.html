How To Use This Playground

It isn't something so complex that it requires an explanation, but I want to write some text to have sample content for this playground.

  • When you enter HTML in the editor below, this preview will be updated based on its content.
  • You can resize the preview by dragging the right and left edges, and resize the editor by dragging the bottom right corner.
  • By adding the dark: prefix to html tag, dark mode is supported.
  • Custom settings are not available. If you want to try it with settings included, I recommend using the official playground (https://play.tailwindcss.com/).
  • When you switch this page to dark mode, the preview will also switch to dark mode.
  • I would appreciate any feedback or advice you have.

Thank you for reading.
